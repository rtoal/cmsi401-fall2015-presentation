Grapevine is...

...An application that automatically populates a user's calendar with events from Twitter and Facebook. Events are extracted by an intelligent agent that listens to feeds that the user follows.

Architecture

APIs

Date Processing

We look for dates in a post or tweet and treat that as an event. We use chrono-node to extract the dates and identify events.

Machine Learning

Trained a multi-label classifier on 2000 tweets and posts. Tags include entertainment, academic, community service, food, etc. About 65% accurate.

Technologies

  • Android SDK
  • Volley - HTTP requests
  • Caldroid - calendar package
